\doxysubsection{Detailed Description}
Abstract Factory class for creating transportation objects. 

The \mbox{\hyperlink{classTransportationFactory}{Transportation\+Factory}} defines an interface for creating various types of \mbox{\hyperlink{classTransport}{Transport}} objects. Derived classes will implement the specific transport creation logic.
