Represents the state where NPCs are in a donation mood. 

This class handles the logic for NPCs when they are willing to donate, affecting the city\textquotesingle{}s economy and happiness levels.

Handles the donation logic for NPCs in this state. 

Handles the logic for NPCs in the donation state.

This method is called to process the actions related to NPC donations, including any impacts on the game state or economy.

This method calculates whether a donation occurs based on the number of NPCs in the donation state and updates the city\textquotesingle{}s funds accordingly.

The chance of receiving an investment increases with the number of NPCs in the \mbox{\hyperlink{classDonationState}{Donation\+State}}. If an investment occurs, the specified amount is added to the city\textquotesingle{}s funds.
