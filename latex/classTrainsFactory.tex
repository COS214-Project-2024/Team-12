\doxysubsection{Detailed Description}
Factory class for creating train transport instances. 

\mbox{\hyperlink{classTrainsFactory}{Trains\+Factory}} is a concrete factory that implements the \mbox{\hyperlink{classTransportationFactory}{Transportation\+Factory}} interface. This class is responsible for creating and returning instances of train transports using the Factory Method pattern.
