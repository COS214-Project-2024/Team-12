\doxysubsection{Detailed Description}
Represents the productive state of NPCs. 

This class handles the logic for NPCs that are in a productive state, including the calculation of productivity boosts based on the number of productive NPCs.

Handles the logic for the productive state of NPCs.

This method calculates the chance of a productivity boost and applies it to the production rate in the \mbox{\hyperlink{classGovernment}{Government}} class if applicable.

This method calculates the chance of a productivity boost based on the number of NPCs in the productive state. If the productivity boost occurs, it increases the production rate in the \mbox{\hyperlink{classGovernment}{Government}} class.
