A concrete implementation of \mbox{\hyperlink{classTransport}{Transport}} representing various types of roads. 

The \mbox{\hyperlink{classRoads}{Roads}} class provides specific implementations for road-\/related attributes like traffic capacity and road type. It implements the \mbox{\hyperlink{classTransport}{Transport}} interface.
