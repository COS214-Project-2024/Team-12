\doxysubsection{Detailed Description}
Abstract class representing a generic transport. 

The \mbox{\hyperlink{classTransport}{Transport}} class defines a prototype for various types of transportation. Derived classes must implement the specific transportation details, such as cost, maintenance cost, speed, capacity, and type. The class also provides a clone method to support creating copies of transport objects.
