\doxysubsection{Detailed Description}
Factory class for creating shop buildings. 

\mbox{\hyperlink{classShopsFactory}{Shops\+Factory}} is a concrete implementation of \mbox{\hyperlink{classCommercialFactory}{Commercial\+Factory}} that creates shop-\/type commercial buildings. This factory is used to encapsulate the creation of shop buildings, adhering to the Abstract Factory design pattern.
