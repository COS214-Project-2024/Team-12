\doxysubsection{Detailed Description}
Abstract base class representing a state in the NPC state machine. 

The \mbox{\hyperlink{classNPCState}{NPCState}} class provides an interface for NPC states, allowing each state to define its own behavior, retrieve its name, and be cloned.
