\doxysubsection{Detailed Description}
Represents a 2D coordinate location on a grid. 

The {\ttfamily \mbox{\hyperlink{structLocation}{Location}}} struct stores x and y coordinates, providing utilities to compare, validate, and calculate the distance between locations.
