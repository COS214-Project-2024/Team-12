\doxysubsection{Detailed Description}
Processes and manages storage for income-\/generating resources. 

The \mbox{\hyperlink{classIncomeResourceProcessor}{Income\+Resource\+Processor}} class handles storing and processing resources that generate income. It maintains a current storage level and a maximum storage capacity. The processed resources contribute their market value to the \mbox{\hyperlink{classGovernment}{Government}} instance.
