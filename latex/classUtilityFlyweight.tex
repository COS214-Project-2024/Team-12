\doxysubsection{Detailed Description}
Abstract base class for utility components, providing shared functionality. 

The \mbox{\hyperlink{classUtilityFlyweight}{Utility\+Flyweight}} class defines shared attributes and methods for various utility types. Implements the Prototype pattern for cloning and defines core utility attributes like cost, capacity, operational status, and resource requirements.
