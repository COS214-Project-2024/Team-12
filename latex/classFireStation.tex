\doxysubsection{Detailed Description}
Represents a fire station in the city. 

The \mbox{\hyperlink{classFireStation}{Fire\+Station}} class is a concrete implementation of the \mbox{\hyperlink{classPublicService}{Public\+Service}} interface. It provides services related to fire safety and emergency response within the city.
