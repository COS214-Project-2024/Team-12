\doxysubsection{Detailed Description}
Abstract base class for processing and managing resources. 

The \mbox{\hyperlink{classResourceProcessor}{Resource\+Processor}} class defines the interface for processing, storing, and displaying resource-\/related information. Derived classes will implement the specific processing and storage logic for different types of resources.
