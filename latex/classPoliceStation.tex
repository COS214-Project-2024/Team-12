\doxysubsection{Detailed Description}
Represents a police station public service in the city. 

Represents a police station, which is a public service in the city.

The \mbox{\hyperlink{classPoliceStation}{Police\+Station}} class inherits from the \mbox{\hyperlink{classPublicService}{Public\+Service}} class and includes utilities required for its operation, along with its status.

The \mbox{\hyperlink{classPoliceStation}{Police\+Station}} class inherits from the \mbox{\hyperlink{classPublicService}{Public\+Service}} class and provides methods to manage its utilities and display its status.
