\doxysubsection{Detailed Description}
Factory class for creating stone resources, providing both income and construction products. 

The {\ttfamily \mbox{\hyperlink{classStoneFactory}{Stone\+Factory}}} class inherits from {\ttfamily \mbox{\hyperlink{classResourceFactory}{Resource\+Factory}}} and overrides methods to create {\ttfamily \mbox{\hyperlink{classIncomeResourceProduct}{Income\+Resource\+Product}}} and {\ttfamily \mbox{\hyperlink{classConstructionResourceProduct}{Construction\+Resource\+Product}}} objects representing stone resources.
