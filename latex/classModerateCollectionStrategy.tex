\doxysubsection{Detailed Description}
Collection strategy for moderate collection rate. 

A strategy for collecting resources at a moderate rate.

Implements a 100\% collection efficiency.

This class provides a method to collect resources at a normal collection rate. It is part of a strategy pattern where different collection strategies can be used based on the game\textquotesingle{}s requirements.
