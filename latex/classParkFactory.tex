\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classPark}{Park}} objects. 

The \mbox{\hyperlink{classParkFactory}{Park\+Factory}} class is responsible for creating instances of the \mbox{\hyperlink{classPark}{Park}} class, allowing for encapsulation of the \mbox{\hyperlink{classPark}{Park}} creation process within the city simulation.
