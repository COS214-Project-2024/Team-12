\doxysubsection{Detailed Description}
A concrete implementation of an \mbox{\hyperlink{classIndustry}{Industry}} for processing metal resources. 

The \mbox{\hyperlink{classMetalWorkFacility}{Metal\+Work\+Facility}} class handles the processing of income-\/generating and construction resources, specifically gold and steel. It maintains a collection range within which it can collect resources.
