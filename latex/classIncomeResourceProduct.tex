\doxysubsection{Detailed Description}
Represents an income-\/generating resource in the city. 

The \mbox{\hyperlink{classIncomeResourceProduct}{Income\+Resource\+Product}} class defines an individual resource that can generate income when processed. Each resource has a name, quantity, market value, and a status indicating if it\textquotesingle{}s ready for collection.
