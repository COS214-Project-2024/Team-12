\doxysubsection{Detailed Description}
Abstract factory class for creating landmark objects. 

The {\ttfamily \mbox{\hyperlink{classLandMarkFactory}{Land\+Mark\+Factory}}} class provides an interface for creating different types of landmarks in the city simulation. Specific landmark types should be created by concrete factories that inherit from this interface.
