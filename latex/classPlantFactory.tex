\doxysubsection{Detailed Description}
A factory class for creating resources related to plants. 

The \mbox{\hyperlink{classPlantFactory}{Plant\+Factory}} class is responsible for creating income-\/generating and construction resources, specifically \mbox{\hyperlink{classCoal}{Coal}} and \mbox{\hyperlink{classWood}{Wood}}.
