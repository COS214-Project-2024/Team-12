\doxysubsection{Detailed Description}
Abstract Factory class for creating residential buildings. 

The \mbox{\hyperlink{classResidentialBuildingFactory}{Residential\+Building\+Factory}} class provides an interface for creating residential building objects. \mbox{\hyperlink{classConcrete}{Concrete}} factories derived from this class will implement the specific creation logic for different types of residential buildings.
